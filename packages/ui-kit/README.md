# @meetgram/ui-kit

All the UI components and icons of Meetgram.

### Usage:

- Install package:

  ```bash
    npm i @meetgram/ui-kit@latest
  ```
  ```bash
    yarn add @meetgram/ui-kit@latest
  ```
  ```bash
    pnpm add @meetgram/ui-kit@latest
  ```
